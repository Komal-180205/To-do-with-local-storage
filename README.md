# 📝 To-Do List App

A simple To-Do List web app that allows users to add and delete tasks. Tasks are stored in **Local Storage**, so they remain even after refreshing the page.

## 🚀 Features

* Add tasks
* Delete tasks
* Data persists using Local Storage

## 🛠️ Tech Used

HTML, CSS, JavaScript

## 🧠 How It Works

* Tasks are stored in `localStorage` as JSON
* On page load, saved tasks are fetched and displayed
* Users can delete tasks dynamically
